# Peak Games

Official website for the Peak Games series — a collection of free browser-based games.

## Games

- **Lucky** — A casino-style experience with dice, mines, roulette, slots, stocks, and more
- **Roguelite Arena** — A roguelite adventure with procedurally generated levels and unique runs every time

## Stack

- HTML, CSS, JavaScript
- No frameworks, no dependencies

## Pages

- `index.html` — Home page with hero, about, and privacy policy
- `games.html` — Games library with search and category filtering
